# סיפוראי - יצירת ספרי ילדים מותאמים אישית עם AI

פלטפורמה אינטראקטיבית שבה ילדים והורים יוצרים ספרי סיפורים מותאמים אישית מאפס באמצעות AI.

## תכונות

- סטודיו יצירת סיפורים מודרך
- איורים מונעי AI לכל עמוד
- מעצב דמויות עם אווטארים AI
- בונה סיפורים סצנה-אחר-סצנה
- כלי AI לסיפורים - ניתוח קצב, בודק נושאים, משפר דיאלוגים
- רב-שפתי - עברית, אנגלית ויידיש עם RTL
- גיימיפיקציה - רמות, XP, תגים, סטריקים
- קהילה - שיתוף ספרים, שיתוף פעולה ותגובות

## טכנולוגיות

| שכבה | טכנולוגיה |
|------|-----------|
| Frontend | React 18 + Vite |
| UI | Radix UI + shadcn/ui + Tailwind CSS |
| AI | LLM (טקסט) + יצירת תמונות (איורים) |
| אנימציה | Framer Motion |
